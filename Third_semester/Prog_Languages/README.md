# Дисциплина «Языки программирования» (2025)

**Дисциплина реализуется в формате воркшопов,на которых проходит знакомство с языками программирования такими как: c, cs - и их функциональностью.**

*Список воркшопов и прикладное назначение выполненной программы:*
- Wokshop_1[] - знакомство с си-подобными языками на примере выполнения шифратора и дешифрратора шифра Цезаря
