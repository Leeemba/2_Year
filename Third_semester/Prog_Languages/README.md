# Дисциплина «Языки программирования» (2025)

**Дисциплина реализуется в формате воркшопов,на которых проходит знакомство с языками программирования такими как: c, c# - и их функциональностью.**

*Список воркшопов и прикладное назначение выполненной программы:*
- [Workshop_1](./workshop1.c) - знакомство с си-подобными языками на примере выполнения шифратора и дешифрратора шифра Цезаря
- [Workshop_2](./workshop2.c) - работа со структурами в си
- [Workshop_3](./Workshop3) - терминальная мини-игра: набор героев с некими характеристиками || организация работа классов в c#
- [Workshop_4](./Workshop4) - обращение к загатовленному датасету и извлечение нужной информации посредством методов,написанных на языку c# 
- [Workshop_5](./Workshop5/Workshop5.cs) - знакомство с асинхронным режимом выполнения программы на языке c# посредством обращения к списку датасетов и делегирования задач по обработке внутренней информации разным потокам
- [Workshop 6](./workshop6) - знакомство с принципом интероперабельности и написание простенькой программы со взаимодействием языков си и python
